This turns out to be pretty simple.

#+BEGIN_EXAMPLE
    $ diskutil list # find the disk you want to eject
    $ diskutil eject /dev/disk2 # eject it
    # wait...
#+END_EXAMPLE

=diskutil eject= by itself will give help on the verb.
